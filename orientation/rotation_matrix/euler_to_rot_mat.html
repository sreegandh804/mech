euler_to_rot_mat¶

ivy_mech.euler_to_rot_mat(euler_angles, convention='zyx', batch_shape=None, dev_str=None)[source]¶

Convert \(zyx\) Euler angles \(\mathbf{θ}_{abc} = [ϕ_a, ϕ_b, ϕ_c]\) to rotation matrix \(\mathbf{R}\in\mathbb{R}^{3×3}\).

[reference]

Parameters

  • euler_angles (array) – Euler angles [batch_shape,3]

  • convention (str, optional) – The axes for euler rotation, in order of L.H.S. matrix multiplication.

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

Rotation matrix [batch_shape,3,3]
