rot_mat_to_euler¶

ivy_mech.rot_mat_to_euler(rot_mat, convention='zyx', f=None)[source]¶

Convert rotation matrix \(\mathbf{R}\in\mathbb{R}^{3×3}\) to \(zyx\) Euler angles \(\mathbf{θ}_{xyz} = [ϕ_z, ϕ_y, ϕ_x]\).

[reference]

Parameters

  • rot_mat (array) – Rotation matrix [batch_shape,3,3]

  • convention (str, optional) – The axes for euler rotation, in order of L.H.S. matrix multiplication.

  • f (ml_framework, optional) – Machine learning framework. Inferred from inputs if None.

Returns

zyx Euler angles [batch_shape,3]
