Convert rotation axis unit vector \(\mathbf{e} = [e_x, e_y, e_z]\) and rotation angle \(θ\) to quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).

[reference]

Parameters

axis_angle (array) – Axis to rotate about and angle to rotate [batch_shape,4]

Returns

Quaternion [batch_shape,4]

ivy_mech.euler_to_quaternion(euler_angles, convention='zyx', batch_shape=None)[source]¶

Convert \(zyx\) Euler angles \(\mathbf{θ}_{abc} = [ϕ_a, ϕ_b, ϕ_c]\) to quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\).

[reference]

Parameters

  • euler_angles (array) – Euler angles, in zyx rotation order form [batch_shape,3]

  • convention (str, optional) – The axes for euler rotation, in order of L.H.S. matrix multiplication.

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

Returns

Quaternion [batch_shape,4]

ivy_mech.get_random_quaternion(max_rot_ang=π, batch_shape=None)[source]¶

Generate random quaternion \(\mathbf{q} = [q_i, q_j, q_k, q_r]\), adhering to maximum absolute rotation angle.

[reference]

Parameters

  • max_rot_ang (float, optional) – Absolute value of maximum rotation angle for quaternion. Default value of \(π\).

  • batch_shape (sequence of ints, optional) – Shape of batch. Shape of [1] is assumed if None.

Returns

Random quaternion [batch_shape,4]
