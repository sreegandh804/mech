Collection of Functions for Homogeneous Co-ordinates

ivy_mech.make_coordinates_homogeneous(coords, batch_shape=None)[source]¶

Append ones to array of non-homogeneous co-ordinates to make them homogeneous.

Parameters

  • coords (array) – Array of non-homogeneous co-ordinates [batch_shape,d]

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

Returns

Array of Homogeneous co-ordinates [batch_shape,d+1]

ivy_mech.make_transformation_homogeneous(matrices, batch_shape=None, dev_str=None)[source]¶

Append to set of 3x4 non-homogeneous matrices to make them homogeneous.

Parameters

  • matrices (array) – set of 3x4 non-homogeneous matrices [batch_shape,3,4]

  • batch_shape (sequence of ints, optional) – Shape of batch. Inferred from inputs if None.

  • dev_str (str, optional) – device on which to create the array ‘cuda:0’, ‘cuda:1’, ‘cpu’ etc. Same as x if None.

Returns

4x4 Homogeneous matrices [batch_shape,4,4]
