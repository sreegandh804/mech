ivy_mech.mat_pose_to_rot_vec_pose(matrix, f=None)[source]¶

Convert matrix pose \(\mathbf{P}\in\mathbb{R}^{3×4}\) to rotation vector pose \(\mathbf{p}_{rv} = [\mathbf{x}_c, \mathbf{θ}_{rv}] = [x, y, z, θe_x, θe_y, θe_z]\).

[reference]

Parameters

  • matrix (array) – Matrix pose [batch_shape,3,4]

  • f (ml_framework, optional) – Machine learning framework. Inferred from inputs if None.

Returns

Rotation vector pose [batch_shape,6]
